1.3 Stacks & Queues

Stacks

A stack can be implemented using an array.

A stack is first in, last out (LIFO).

The INSERT operation on a stack is often called PUSH, and the DELETE operation is often called POP.

Queues

A queue can be implemented using an array.

A queue is first in, first out (FIFO).

The INSERT operation on a queue is called ENQUEUE, and the DELETE operation is often called DEQUEUE.

A queue has a head and a tail.
